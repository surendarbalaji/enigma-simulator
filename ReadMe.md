# enigma simulator

a simple enigma I simulator

![](https://github.com/surendarbalaji/enigma-simulator/blob/main/enigma.gif)

## features
* **the enigma machine** - you literally just type letters and it gets funnelled through the enigma machine. i think it just looks cool more than anything else
* **path visualisation** - yeah i guess this is a feature if you really squint your eyes, because this is kind of the whole point of making a simulator. though mine is pretty cool in that i decided to have the inner rotor connections be completely straight and have it zig-zag in the spaces in-between, which isn't actually how the enigma machine works, and it made is 3x more difficult to debug, however i still think it looks cool.

## controls

* type letters
